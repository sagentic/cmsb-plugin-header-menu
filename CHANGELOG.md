# Changelog

All notable changes to the Header Menu Admin plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.00] - 2026-01-10

### Added
- Initial release
- Add custom links to header menu
- Remove existing links with pattern matching
- Support for contains, exact match, and regex matching
- Visibility control (always, logged in, admin only)
- Target window selection (same window, new tab)
- Position control (start or end of menu)
- Enable/disable toggle for individual rules
- Dashboard with rule list
- Edit and delete functionality
- Built-in help page with comprehensive documentation
- Input validation for all dropdown/select fields
- ReDoS protection for regex pattern matching
- Screen reader text for accessibility on icon-only buttons
- Form label associations using id/for attributes
- ARIA attributes for help text and error messages
- Success flash messages for save, delete, and toggle actions
- Validation error display with clear messaging
- Required field indicators with red asterisks
- Contextual help text for all form fields
- README.md documentation
- CHANGELOG.md version history
- QUICK_START.md installation guide
- LICENSE file (MIT)
- reset_installation.php tool
